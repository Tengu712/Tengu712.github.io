# Layout

## basic

本ホームページの基本レイアウト。

- ヘッダーがある
- 中央にコンテンツがある
- フッターがある

パラメータ:

- `title`: ページタイトル

## post

記事専用レイアウト。

- ヘッダーがある
- 中央にコンテンツがある
  - コンテンツの先頭に次が並ぶ
     1. ジャンル別アイコン
     2. ページタイトル (h1)
     3. ジャンル・タグ・執筆年月日
  - コンテンツの末尾に墓石がある
- 右に目次がある
- フッターがある

パラメータ:

- `title`: ページタイトル
- `genre`: ジャンル
- `tags`: タグ
- `date`: 執筆年月日`YYYY/MM/DD`
